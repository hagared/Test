# ДОМ — браузерный 3D хоррор

Маленькая первоперсонажная страшилка на Three.js. Тёмный лабиринт, фонарик,
ищешь 5 страниц, а за тобой кто-то ходит.

**Предупреждение:** громкие звуки, скримеры, мигающий свет.
Не играй, если у тебя проблемы со здоровьем (эпилепсия, сердце, тревожные расстройства).

## Управление
- `W A S D` — двигаться
- `Мышь` — смотреть (Pointer Lock)
- `Shift` — бежать (тратит выносливость)
- `F` — фонарик вкл/выкл
- `Esc` — пауза

## Как запустить локально

Нужен любой статический HTTP-сервер (модули ES не работают через `file://`):

```bash
cd horror-3d
python3 -m http.server 8080
# открой http://localhost:8080
```

## Технические детали

- Чистый ES-модульный JS, без сборки. Three.js лежит локально в `vendor/three/` и подключается через import-map — никаких внешних CDN.
- Вся музыка/звуки сгенерированы программно через WebAudio API — никаких
  внешних аудиофайлов. Это значит игра самодостаточна и быстро грузится.
- Скример рисуется на canvas в момент срабатывания (тоже без ассетов).

## Структура

```
horror-3d/
├── index.html
├── styles.css
└── js/
    ├── main.js     # сцена, игровой цикл, склейка
    ├── maze.js     # генерация лабиринта + коллизии
    ├── monster.js  # модель и поведение монстра (Weeping-Angel + телепорт)
    ├── audio.js    # WebAudio: дрон, шёпот, сердцебиение, скрим
    └── ui.js       # HUD, оверлеи, рисование скримера
```
